# our custom category ints

used for our own purposes will start at 0 and then use negative numbers for new categories we create
```
0  Return all items
-1 Items sold by vendors
-2 'supply and provision' # https://ffxiv.gamerescape.com/wiki/Supply_and_Provisioning_Mission
-3 'crafter quest item'  # https://ffxivgillionaire.com/crafter-class-quest-items-guide
-4 NPC Vendor Furniture
-5 Current Crafted Gear (used to exclude it from glamor searches)
-666 Gil Laundering Items (exclude items used for gil laundering more often than actual sales)
```
